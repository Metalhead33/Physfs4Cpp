# Physfs4Cpp
C++ wrapper for [PhysFS][1].

[1]: http://icculus.org/physfs

Works with both Cmake and Qmake.

Requires PhysFS, obviously.

Special thanks to [Kevin Howell][2] for creating [PhysFS++][3], the primary inspiration for this project.
